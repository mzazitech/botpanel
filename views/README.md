# Mzazi Tech Store 🚀

A premium full-stack digital marketplace for WhatsApp Bots, Pterodactyl Panels, VPS, and API Keys with Paystack integration.

## Features
- 🔐 JWT Authentication & bcrypt password hashing
- 💳 Paystack payment gateway (KES currency)
- 📦 Automatic API key delivery on successful payment
- 👑 Complete Admin Panel (users, products, payments)
- 🌓 Light/Dark mode toggle
- 🔍 Product search
- 📱 Fully responsive cyberpunk-themed UI
- 🛡️ Security: Helmet, rate limiting, input validation

## Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose ODM)
- **Frontend:** EJS, Custom CSS (Glassmorphism + Neon)
- **Payment:** Paystack API
